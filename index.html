03

Kalau diminta login atau cookies

Gunakan cookies dari browser dan akunmu sendiri. Jangan mengunggah cookies ke GitHub.

Cara paling aman

Ambil langsung dari browser

Tutup browser terlebih dahulu bila terjadi error, lalu ganti nama browser sesuai yang kamu pakai.

Chromeyt-dlp --cookies-from-browser chrome "TAUTAN_VIDEO"
Edgeyt-dlp --cookies-from-browser edge "TAUTAN_VIDEO"
Firefoxyt-dlp --cookies-from-browser firefox "TAUTAN_VIDEO"

Pakai file cookies.txt

Simpan file di folder Downloads. Pakai hanya file milikmu sendiri dan jangan pernah membagikannya.

Windows
yt-dlp --cookies "%USERPROFILE%\Downloads\cookies.txt" "TAUTAN_VIDEO"
Linux/macOS
yt-dlp --cookies "$HOME/Downloads/cookies.txt" "TAUTAN_VIDEO"
Buka file referensi cookies ↗
Penting: cookies dapat membuka sesi akun. Jangan unggah ke repository, kirim ke orang lain, atau memakai cookies milik orang lain.

05

Cari tulisan error yang muncul

yt-dlp tidak dikenaliPerintah belum terpasang atau terminal belum dibuka ulang⌄

Tutup terminal, buka kembali, lalu jalankan yt-dlp --version. Bila tetap gagal, ulangi langkah pemasangan sesuai perangkatmu.

ffmpeg not foundFFmpeg belum terpasang⌄

Buka panduan perangkat, salin kembali perintah pemasangan, lalu cek dengan ffmpeg -version.

Sign in to confirm you’re not a botGunakan cookies dari browser sendiri⌄

Tutup browser, lalu tambahkan --cookies-from-browser chrome, edge, atau firefox pada perintah.

Permission denied di Android⌄

Jalankan termux-setup-storage, tekan Izinkan, kemudian ulangi. Periksa juga izin penyimpanan Termux di Pengaturan Android.

Requested format is not available⌄

Pilih kualitas “Terbaik yang tersedia” pada pembuat perintah. Jangan memaksa 1080p bila video aslinya hanya tersedia 720p.

HTTP Error 403 / 429⌄

Perbarui yt-dlp, tunggu beberapa saat, matikan VPN bila ada, lalu coba lagi. Terlalu banyak percobaan dalam waktu singkat dapat memicu pembatasan sementara.

Versi yt-dlp terlalu lama⌄

Untuk binary resmi jalankan yt-dlp -U. Untuk Winget, APT, DNF, Pacman, APK, atau Homebrew, perbarui memakai package manager yang sama.

Error belum ada di daftar. Coba cari dengan satu atau dua kata saja.
