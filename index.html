ytconv error? Coba cara ini.

Unduh video atau audio lewat CMD di Windows, Termux di Android, atau buka pilihan web cadangan. Semua langkah dibuat singkat dan bisa langsung disalin.

Buat perintah unduhan Mulai dari Windows
Pakai dengan wajar.

Unduh hanya konten milik sendiri, konten bebas pakai, atau konten yang sudah diizinkan pemiliknya. Jangan dipakai untuk melewati konten berbayar atau perlindungan akses.

01

Buat perintah tanpa mengetik sendiri

Tempel tautannya, pilih hasil yang diinginkan, lalu salin perintah.

Perintah untuk Windows CMD

File akan masuk ke folder Downloads Windows.

02

Cara paling mudah di Windows

Cukup pasang dua alat sekali saja: yt-dlp untuk mengambil file dan FFmpeg untuk menyatukan video dengan suara.

1

Buka CMD sebagai Administrator

Tekan tombol Windows, ketik CMD, lalu pilih Run as administrator.

2

Pasang yt-dlp dan FFmpeg

Salin dua baris ini ke CMD. Tunggu sampai masing-masing selesai.

CMD
winget install --id yt-dlp.yt-dlp --exact --accept-source-agreements --accept-package-agreements
winget install --id Gyan.FFmpeg --exact --accept-source-agreements --accept-package-agreements
Setelah selesai, tutup CMD lalu buka lagi supaya perintahnya terbaca.
3

Periksa apakah sudah terpasang

Kalau muncul nomor versi, berarti sudah siap.

CMD
yt-dlp --version
ffmpeg -version
4

Mulai mengunduh

Ganti TAUTAN_VIDEO dengan tautan yang ingin diunduh.

Video MP4 kualitas terbaik
yt-dlp -f "bv*+ba/b" --merge-output-format mp4 -o "%USERPROFILE%\Downloads\%(title)s.%(ext)s" "TAUTAN_VIDEO"
Audio MP3
yt-dlp -x --audio-format mp3 --audio-quality 0 -o "%USERPROFILE%\Downloads\%(title)s.%(ext)s" "TAUTAN_VIDEO"
Semua video dalam playlist
yt-dlp -f "bv*+ba/b" --merge-output-format mp4 -o "%USERPROFILE%\Downloads\%(playlist_title)s\%(playlist_index)02d - %(title)s.%(ext)s" "TAUTAN_PLAYLIST"

Kalau mendadak error, perbarui dulu

Situs video sering berubah. Versi terbaru biasanya membawa perbaikan.

03

Kalau diminta login atau dianggap bot

Cookies membuat yt-dlp memakai sesi browser milikmu sendiri. Ini hanya diperlukan pada beberapa video.

Cookies sama pentingnya dengan kunci akun

Jangan kirim file cookies ke orang lain, jangan unggah ke GitHub, dan jangan memakai cookies milik orang lain. Cara yang paling aman adalah membaca cookies langsung dari browser sendiri.

Disarankan

Ambil langsung dari browser

Pastikan kamu sudah login di browser. Tutup browser kalau muncul pesan file sedang dipakai, lalu jalankan salah satu perintah ini.

Chromeyt-dlp --cookies-from-browser chrome "TAUTAN_VIDEO"
Edgeyt-dlp --cookies-from-browser edge "TAUTAN_VIDEO"
Firefoxyt-dlp --cookies-from-browser firefox "TAUTAN_VIDEO"

Kalau sudah punya cookies.txt

Simpan file milikmu sendiri di folder Downloads, lalu pakai perintah berikut.

CMD
yt-dlp --cookies "%USERPROFILE%\Downloads\cookies.txt" "TAUTAN_VIDEO"

Setelah selesai, hapus file cookies jika sudah tidak diperlukan.

Tautan Drive yang dikirim sebagai referensi

Isi file pada tautan ini belum dapat diperiksa dari proyek ini. Buka hanya jika kamu mengenali sumbernya. Jangan jalankan file yang mencurigakan dan jangan gunakan cookies akun milik orang lain.

04

Cara di Android lewat Termux

Tidak perlu root. Siapkan ruang kosong karena video akan disimpan ke folder Download.

$ pkg update $ pkg install $ yt-dlp...
Android 7 atau lebih baru

Gunakan Termux dari sumber resminya

Versi F-Droid atau halaman GitHub resmi lebih cocok untuk paket lengkap. Jangan mencampur Termux dan pluginnya dari sumber berbeda.

Buka F-Droid GitHub resmi
1

Perbarui Termux dan pasang alatnya

Termux
pkg update -y && pkg upgrade -y
pkg install python ffmpeg nodejs -y
python -m pip install -U "yt-dlp[default]"
2

Izinkan akses ke folder Download

Jalankan perintah ini, lalu tekan Izinkan saat Android bertanya.

Termux
termux-setup-storage
3

Unduh video atau audio

Video ke folder Download
yt-dlp -f "bv*+ba/b" --merge-output-format mp4 -P "/sdcard/Download" "TAUTAN_VIDEO"
Audio MP3 ke folder Download
yt-dlp -x --audio-format mp3 --audio-quality 0 -P "/sdcard/Download" "TAUTAN_VIDEO"
Ambil pembaruan terbaru
python -m pip install -U --pre "yt-dlp[default]"

Cobalt

Dalam panduan ini, gunakan untuk TikTok dan layanan media sosial lain yang didukung. Bukan pilihan utama untuk YouTube di halaman ini.

Buka Cobalt

Jangan masukkan akun atau kata sandi ke situs unduh pihak ketiga. Kalau situs meminta login, tutup halaman dan gunakan yt-dlp dengan cookies browser milik sendiri.

06

Error yang paling sering muncul

'yt-dlp' is not recognizedWindows tidak menemukan yt-dlp

Tutup CMD, buka lagi, lalu coba yt-dlp --version. Kalau masih sama, ulangi pemasangan dengan winget dan pastikan prosesnya selesai tanpa tulisan gagal.

ffmpeg not foundVideo dan suara tidak bisa disatukan

Pasang FFmpeg, lalu tutup dan buka kembali CMD. Periksa dengan ffmpeg -version.

Sign in to confirm you're not a botPerlu sesi browser milikmu

Perbarui yt-dlp. Kalau belum berhasil, login di Chrome lalu gunakan --cookies-from-browser chrome. Tutup Chrome bila cookies tidak bisa dibaca.

Requested format is not availableKualitas yang dipilih tidak tersedia

Coba perintah paling sederhana terlebih dahulu. yt-dlp akan memilih format yang tersedia secara otomatis.

HTTP Error 403: ForbiddenAkses ditolak sementara

Perbarui yt-dlp, matikan VPN bila sedang aktif, lalu coba lagi. Pada video tertentu, cookies browser mungkin diperlukan.

Permission denied di AndroidTermux belum diizinkan membuka penyimpanan

Jalankan termux-setup-storage, tekan Izinkan, lalu ulangi unduhan. Periksa juga izin Termux di Pengaturan Android.

No space left on deviceRuang penyimpanan sudah penuh

Hapus atau pindahkan beberapa file terlebih dahulu. Video berkualitas tinggi bisa membutuhkan ruang lebih besar karena video dan audio diunduh terpisah sebelum disatukan.

Tidak ada error yang cocok. Coba kata yang lebih pendek.

07

Pasang website ini ke GitHub dan Vercel

Tidak perlu Codex, terminal, atau build. Cukup unggah file sesuai nama dan foldernya.

Struktur fileSiap unggah
unduh-lagi-vercel/
├── index.html
├── style.css
├── script.js
├── vercel.json
├── README.md
└── assets/
    └── favicon.svg
1

Buat repository baru di GitHub

Beri nama misalnya unduh-lagi. Pilih Public atau Private sesuai kebutuhan.

2

Unggah semua file

Pilih Add file → Upload files. Pastikan folder assets tetap ada dan berisi favicon.svg.

3

Hubungkan ke Vercel

Di Vercel pilih Add New → Project, pilih repository tadi, lalu tekan Deploy. Tidak perlu mengisi Build Command.

4

Selesai

Setiap kali file di GitHub diperbarui, Vercel akan memasang versi baru secara otomatis.
