Mudah
1

Buka CMD

Tekan Start, ketik CMD, lalu buka.

2

Pasang yt-dlp dan FFmpeg

Salin lalu jalankan.

CMD
winget install yt-dlp.yt-dlp
winget install Gyan.FFmpeg
3

Mulai mengunduh

Ganti TAUTAN_VIDEO dengan tautan videomu.

Contoh MP4
yt-dlp -f "bv*+ba/b" --merge-output-format mp4 -P "%USERPROFILE%\Downloads" "TAUTAN_VIDEO"

File masuk ke folder Downloads.

02

Buat perintah

Tempel tautan. Perintah berubah otomatis.

Perintah untuk Windows CMD
yt-dlp -f "bv*+ba/b" --merge-output-format mp4 -P "%USERPROFILE%\Downloads" "TAUTAN_VIDEO"

File masuk ke folder Downloads Windows.

Cari error

yt-dlp tidak dikenaliAlat belum terpasang

Ulangi langkah pemasangan sesuai perangkat, lalu tutup dan buka terminal.

ffmpeg not foundFFmpeg belum terpasang

Salin kembali perintah pemasangan FFmpeg dari panduan perangkat.

403 / login diperlukanGunakan cookies akunmu

Pasang extension cookies, ekspor sebagai Netscape, lalu pilih cookies.txt di pembuat perintah.

Android tidak bisa menyimpanIzin penyimpanan belum aktif

Jalankan termux-setup-storage, tekan Izinkan, lalu pakai folder $HOME/storage/downloads. Itu adalah Penyimpanan internal → Download.
